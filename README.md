## How to run? 
### STEP 01- Create a conda environment after opening the repository
```bash
conda create -p insuranceCostEnv python -y
```

```bash
source activate ./insuranceCostEnv
```

### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```

```bash
python app.py
```
